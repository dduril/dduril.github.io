Favorite Books

Currently, I am working through a collection of programming books published by Murach. The goal is to keep my coding skills up to date by reading and coding short programs in Java, Python, and MySQL. Working through these books is a good refresher and opportunity to run/type-up code samples, building a bootcamp-style collection of snippets and reference material.

Currently Reading

  • Murach's Python
  • Murach's MySQL, 2nd Edition

Big Data/Data Analytics

This is a good collection of books, some for reading cover-to-cover, others, like the Cookbooks, are ideal for looking up specific skills and techniques. I tend to favor the books by O'Reilly, but have also found a lot of good books from Apress, Manning, Murach and Packt Publishing. Ofcourse, there are a lot of very good resources available online - blogs, wikis, language documentation, github projects, stackoverflow code samples and vendor provided e-books.

  • Learning R (O'Reilly)

  • Web Scraping with Python (O'Reilly)
