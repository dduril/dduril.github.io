Front End Developer

If you'd like to be a front end developer - here is a list of what you need to know. Learn these items well.

This list is by no means comprehensive, but serves to provide a good starting place for the main topics and technologies.

There are plenty of wonderful online resources, blogs, newsletters, tutorials and opinion on all of these topics.

Testing Frameworks
