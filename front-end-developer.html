Front End Developer

If you'd like to be a front end developer - here is a partial list of what you need to know. Learn these items well.

This list is by no means comprehensive. New items are added all the time, and existing items change and take on new shapes. This list only serves to provide a jumping off place for the modern day Front End or Full Stack Web Developer.

There are plenty of online resources (Github, Stack Overflow), blogs, wikis, newsletters, tutorials and opinion on all of these topics.
